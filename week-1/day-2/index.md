## Daily JS Take 1

Variable assignment and primitive data types

```js
var x = 1;
var y = 'abc';
var z = true;
var a = x;
x = 2;

console.log(x); // 2
console.log(y); // "abc"
console.log(z); // true
console.log(a); // 1
```
